# AI 数据与文档分析助手

这是一个基于 Streamlit 的智能分析应用，支持上传 Excel/CSV 数据表格或 Word 文档，利用 GPT-4o 或 Claude 3.5 自动生成专业的中文分析报告。

## 功能特点
1. **多格式支持**：
   - **数据分析**：支持 `.xlsx` 和 `.csv` 文件，自动进行统计分析和图表建议。
   - **文档分析**：支持 `.docx` 文件，自动进行内容摘要、关键信息提取和深度解读。
2. **智能预览**：
   - 表格数据：展示前 10 行及统计描述。
   - 文档内容：展示前 1000 字符文本预览。
3. **自然语言查询**：您可以输入任意分析需求，如“分析销售趋势”、“总结核心观点”。
4. **双模型支持**：内置 OpenAI (GPT-4o) 和 Anthropic (Claude 3.5) 接口。
5. **专业报告生成**：
   - 数据报告：包含数据概览、关键发现、可视化建议和业务建议。
   - 文档报告：包含内容摘要、关键信息提取、深入分析和总结建议。
6. **一键导出**：支持下载生成的 Markdown 报告。

## 快速开始

### 1. 安装依赖

确保您已安装 Python 3.8+。在终端中运行以下命令安装所需库：

```bash
pip install -r requirements.txt
```

### 2. 运行应用

在终端中运行：

```bash
streamlit run app.py
```

### 3. 使用说明
1. **配置 API Key**：在左侧侧边栏选择模型提供商（OpenAI 或 Anthropic），并输入您的 API Key。
   - 如果使用国内中转服务，请填写 Base URL（例如 `https://api.openai-proxy.com/v1`）。
2. **上传文件**：点击主界面的“上传文件”按钮，选择您的 Excel、CSV 或 Word 文件。
3. **输入需求**：在文本框中输入您的分析目标，或直接使用默认提示词。
4. **生成报告**：点击“开始分析”按钮，等待 AI 生成报告。
5. **下载报告**：生成完成后，点击“下载报告”保存结果。

## 注意事项
- 请确保您的 API Key 具有相应的模型访问权限（如 `gpt-4o` 或 `claude-3-5-sonnet`）。
- 对于非常大的 Word 文档，系统会截取前 10000 字符进行分析，以符合模型 Token 限制。
